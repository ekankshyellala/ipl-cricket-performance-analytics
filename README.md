# ipl-cricket-performance-analytics
Analyzed 12 seasons of IPL cricket data (260K+ ball-by-ball records, 1K+ matches) using Python, SQL, Excel &amp; Power BI.  Built interactive dashboard tracking player performance, team wins, venue statistics &amp; season trends.  Key findings: V Kohli leads with 8.3K+ runs, YS Chahal #1 bowler with 213 wickets, Mumbai Indians most dominant team.
